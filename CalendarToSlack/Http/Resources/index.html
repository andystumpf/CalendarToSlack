Calendar to Slack

Marks you "Away" in Slack during events on your Outlook calendar.

What it does

When an event is happening on your Outlook calendar, Calendar to Slack updates your Slack presence to "Away" and optionally appends a status message (e.g. | Away) to your last name.

Features

Status messages in last name.

If the full-access token is provided, status messages are appended to your last name, so your full name displays as something like Rob Hruska | Away.

Prerequisite: you'll have to give your personal, full-access Slack auth token to the server administrator. Contact them and discuss the security implications of that.

Whitelist support for status messages.

Example: Lunch|1:1|Standup|Meeting|Planning>Meeting

If any of these tokens are found in the calendar event subject, the token will be used as the status message instead of the default "Away".

A > maps matched values on the left hand side to the status message on the right hand side.

Conditional status based on OOO/Busy/Tentative/etc.

OOO and Busy events will set the status message and mark you as "Away". Tentative and Working Elsewhere events will set the status message, but keep your Slack status "Auto" (essentially "Available").

Tips & Tricks

  • In a different office for the week? Add an all-week Working Elsewhere event to your calendar named "London" and add "London" to your whitelist.
  • Want to stay available during lunch but let people know you're only partially around? Create a recurring 1-hour event named "Lunch" that's Tentative.
